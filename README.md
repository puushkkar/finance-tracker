# Personal Finance Tracker

A simple, secure personal finance tracker to manage your income, expenses, and savings. Built entirely with HTML, CSS, and JavaScript.

## Features

- **Dashboard** - View your total income, expenses, and balance at a glance
- **Add Transactions** - Track income (salary, gifts) and expenses (food, bills, shopping)
- **Transaction History** - See all your transactions with dates and descriptions
- **Delete Transactions** - Remove any transaction you no longer need
- **Auto-Save** - Your data is automatically saved in your browser
- **Responsive Design** - Works on mobile, tablet, and desktop

## How to Use

### Option 1: Run Locally
1. Download this repository (click the green **Code** button > **Download ZIP**)
2. Extract the ZIP file
3. Open `index.html` in any web browser (Chrome, Firefox, Edge, etc.)
4. Start tracking your finances!

### Option 2: Use GitHub Pages (Hosted Version)
1. Go to repository **Settings** > **Pages**
2. Under **Build and deployment**, select **Deploy from a branch**
3. Choose branch **main** and folder **/ (root)**
4. Click **Save**
5. Your app will be live at: `https://puushkkar.github.io/finance-tracker/`

## Security

- **No server** - Everything runs in your browser
- **No database** - Data stored in browser localStorage only
- **No API calls** - No data is sent to any server
- **No external libraries** - Pure vanilla JavaScript
- **No login required** - No credentials to steal
- **100% safe** - Cannot be hacked or used to access your computer

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
